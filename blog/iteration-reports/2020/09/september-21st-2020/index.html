Monday, September 21, 2020

Report on iteration Sep 7, 2020 — Sep 18, 2020

This iteration included work to complete cdh-web version 2.7, which consisted of maintenance upgrades and a few helpful features for fall; maintenance and bug fixes on PPA, including a refactor to use in-house Solr library parasolr, removing a dependency on a forked, unmaintained SolrClient; and continued progress on Startwords.

Major work not reflected here includes the final revisions on charter for the CDH/Geniza research partnership, which was signed and approved today, and improved planning and documentation for the steps to close out a project when a research partnership grant ends.

Demos

  • CDH web affiliates page now includes staff, such as PUL librarians directing data curation projects
  • CDH web events can now be designated as virtual, and take an optional URL for joining the virtual event
